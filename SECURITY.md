## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an email to the repository owner. We consider security vulnerabilities to be a high priority and will work to resolve them as quickly as possible.

Please do not open a public issue for security vulnerabilities. Instead, wait for a response via email.

When reporting a vulnerability, please include the following information:

*   A description of the vulnerability.
*   Steps to reproduce the vulnerability.
*   Any potential impact or risks associated with the vulnerability.
*   Any suggestions for fixing the vulnerability.

We will review your report and respond as soon as possible. Thank you for helping to keep this project secure!
